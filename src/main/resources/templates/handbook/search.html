Загрузите файл для поиска

Загрузите CSV или Excel файл, содержащий ID товаров и штрихкоды/наименования. Система найдёт соответствующие ссылки в справочнике.

Перетащите файл сюда или нажмите для выбора

Требования к файлу

Файл должен содержать строку заголовков. Поддерживаемые форматы: CSV, XLSX, XLS.

Обязательные колонки:

  • Колонка с идентификатором строки (любое название, выбирается при настройке)

Дополнительные колонки (хотя бы одна из двух):

  • Штрихкод товара
  • Наименование товара
