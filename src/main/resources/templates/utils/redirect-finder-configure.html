Параметры обработки
Максимальное время ожидания ответа от одного URL.
• Меньше время: быстрее обработка, но могут пропадать медленные сайты
• Больше время: больше шансов найти ссылку, но общая обработка дольше
Пауза между запросами для избежания блокировки.
• Без задержки: максимальная скорость, но риск блокировки
• 0.5-2 секунды: оптимально для большинства сайтов
• 5+ секунд: для строгих анти-бот систем (напр. Wildberries, ОЗОН)
Использовать Playwright вместо curl для всех URL
Стратегии обработки
1. cURL
Основная стратегия, быстро и эффективно
2. Playwright
При блокировке cURL, использует браузер
3. HttpClient
Резервная стратегия
Режимы обработки:
  • В фоне: Для больших файлов. Можете заниматься другой работой, уведомление придёт автоматически
  • Быстро: Для файлов до 50 URL. Файл скачается сразу после обработки
