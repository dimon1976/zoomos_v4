К утилитам

Поиск финальных ссылок после редиректов

Утилита обрабатывает CSV/Excel файлы с URL для получения финальных ссылок после HTTP редиректов (301, 302, 307, 308).

3 стратегии обхода блокировок
Обработка до 20 редиректов
Детальная статистика
Шаг 1: Загрузка файла
Выберите файл с URL для обработки

Поддерживаются форматы: Excel (.xlsx, .xls), CSV (.csv)

Требования к файлу

  • Обязательная колонка с URL
  • Колонки ID и Модель опциональны
  • URL должны начинаться с http:// или https://
  • Поддержка до 20 редиректов на URL
  • Максимальный размер файла: 1200MB

Пример обработки

Исходные данные:
ID: 12345
URL: https://short.ly/abc123
Модель: Товар А
Результат:
ID: 12345
Финальный URL: https://shop.example.com/product/12345
Редиректов: 2 | Статус: OK | Стратегия: curl

Важно знать

  • Обработка может занять много времени
  • Некоторые сайты блокируют запросы
  • Автоматическое переключение стратегий
  • Указание использованной стратегии
