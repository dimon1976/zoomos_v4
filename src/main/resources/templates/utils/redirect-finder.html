Стратегии защиты от антибот систем

Шаг 1: Загрузка файла
Выберите файл с URL для обработки

Поддерживаются форматы: Excel (.xlsx, .xls), CSV (.csv)

Требования к файлу

  • Файл должен содержать колонку с URL (обязательно)
  • Колонки ID и Модель опциональны
  • URL должны начинаться с http:// или https://
  • Поддерживается до 20 редиректов на URL
  • Максимальный размер файла: 1200MB

Пример обработки

Исходные данные:
ID: 12345
URL: https://short.ly/abc123
Модель: Товар А
Результат:
ID: 12345
Финальный URL: https://shop.example.com/product/12345
Редиректов: 2 | Статус: OK | Стратегия: curl

Важно знать

  • Обработка большого количества URL может занять значительное время
  • Некоторые сайты могут блокировать автоматические запросы
  • Для заблокированных URL автоматически используется браузерная стратегия
  • В результате указывается какая стратегия была использована
