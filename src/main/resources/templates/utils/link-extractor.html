К утилитам

Сбор ссылок с привязкой к ID

Утилита автоматически сканирует все колонки файла и собирает все найденные URL с привязкой к ID.

Множественные ссылки в ячейке
Определение колонки
Экспорт в CSV и Excel
Шаг 1: Загрузка файла
Выберите файл для обработки

Поддерживаются форматы: Excel (.xlsx, .xls), CSV (.csv), TXT (.txt)

Требования к файлу

  • Обязательная колонка с ID
  • Ссылки могут находиться в любых колонках
  • Поддержка ссылок http:// и https://
  • Несколько ссылок в одной ячейке
  • Максимальный размер файла: 1200MB

Как это работает

Загрузка
Загрузите Excel или CSV файл
Настройка
Выберите колонку ID
Поиск
Поиск ссылок во всех колонках
Результат
Пары "ID - ссылка"

Пример обработки

Исходные данные:
ID: 12345
Описание: Товар https://example.com/item1
Ссылка: https://shop.ru/product/123
Результат:
12345 | https://example.com/item1 | Колонка 2
12345 | https://shop.ru/product/123 | Колонка 4

Возможности

  • Поддержка множественных ссылок в одной ячейке
  • Определение колонки с найденной ссылкой
  • Экспорт в CSV и Excel
  • Обработка больших файлов
