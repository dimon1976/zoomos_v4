К утилитам
Сбор ссылок с привязкой к ID

Утилита автоматически найдет все ссылки в файле и свяжет их с соответствующими ID. Поиск ведется по всем колонкам в каждой строке.

1. Загрузка

Загрузите Excel или CSV файл с данными

2. Настройка

Выберите колонку с идентификаторами

3. Поиск

Автоматический поиск ссылок во всех колонках

4. Результат

Файл с парами "ID - найденная ссылка"

Шаг 1: Загрузка файла
Выберите файл для обработки

Поддерживаются форматы: Excel (.xlsx, .xls), CSV (.csv), TXT (.txt)

Требования к файлу
  • Файл должен содержать колонку с идентификаторами (ID)
  • Ссылки могут находиться в любых колонках
  • Поддерживаются ссылки вида http:// и https://
  • В одной ячейке может быть несколько ссылок
  • Максимальный размер файла: 1200MB
Что произойдет с данными
Исходные данные:
ID: 12345
Описание: Товар https://example.com/item1
Цена: 1000 руб.
Ссылка: https://shop.ru/product/123
Результат:
12345 | https://example.com/item1 | Колонка 2
12345 | https://shop.ru/product/123 | Колонка 4
