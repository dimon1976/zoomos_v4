Назад к утилитам

Обработка файлов статистики

Утилита обрабатывает файлы статистики с возможностью выбора колонок и изменения данных для создания кастомизированных отчетов.

Выбор колонок чекбоксами
Замена данных
Множественная загрузка
Экспорт в Excel/CSV
Шаг 1: Загрузка файлов статистики
Выберите файлы статистики для обработки

Поддерживаемые форматы: Excel (.xlsx, .xls), CSV (.csv), JSON (.json)

Требования и возможности

Обязательно
  • Файлы типа hoff.ru-sites-mapping-*.xlsx
  • 39 колонок со структурой данных
  • Форматы: Excel, CSV, JSON
Возможности
  • Выбор колонок чекбоксами
  • Замена источника для клиента
  • Множественная загрузка файлов
  • Экспорт в Excel и CSV
  • Файлы до 600MB

Как это работает

1
Загрузите файлы

Excel, CSV или JSON файлы со статистикой

2
Выберите колонки

отметьте нужные колонки чекбоксами из 39 доступных

3
Замените данные

укажите новые значения для источника или других полей

4
Скачайте отчет

кастомизированный файл в нужном формате

Пример обработки

Исходные данные:
39 колонок: домен, ID, товары, конкуренты
Результат:
Выбранные колонки + измененные данные для клиента

Возможности

  • Множественная загрузка файлов
  • Интерактивный выбор колонок
  • Настройка замены источника
  • Экспорт в различных форматах
