Обработка файлов статистики

Утилита обрабатывает файлы статистики с возможностью выбора колонок и изменения данных для создания кастомизированных отчетов.

Шаг 1: Загрузка файлов статистики
Выберите файлы статистики для обработки

Поддерживаемые форматы: Excel (.xlsx, .xls), CSV (.csv), JSON (.json)

Требования к файлам

  • Файлы типа hoff.ru-sites-mapping-*.xlsx
  • 39 колонок с данными о товарах
  • Домен, ID сайта, товары, конкуренты
  • Выбор нужных колонок чекбоксами
  • Замена источника для клиента
  • Максимальный размер файла: 600MB
