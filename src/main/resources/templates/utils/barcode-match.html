Назад к утилитам

Сопоставление штрихкодов

Утилита находит соответствия между штрихкодами в одном файле и создает результирующий файл с парами ID-штрихкод-URL.

Поддержка множественных штрихкодов через запятую
Валидация 13-символьных штрихкодов
Экспорт в Excel формат
Шаг 1: Загрузка файла
Перетащите файл сюда или нажмите для выбора

Поддерживаемые форматы: Excel (.xlsx, .xls), CSV (.csv)

Требования к файлу

Структура файла
  • Первая строка - заголовки колонок
  • Колонка с ID записей
  • Колонка со штрихкодами
  • Колонка с URL
Формат штрихкодов
  • 13 символов (цифры)
  • Несколько ШК через запятую: "1234567890123,9876543210987"
  • Неверные ШК будут пропущены
Результат
  • Excel файл с тремя колонками
  • ID | Штрихкод | URL
  • Отдельная строка для каждого ШК
