Очистка URL

Утилита удаляет UTM-метки, реферальные и отслеживающие параметры из ссылок, оставляя чистые URL.

Поддерживаемые форматы: Excel (.xlsx, .xls), CSV (.csv), TXT (.txt)

Требования к файлу

  • Первая строка - заголовки колонок
  • Обязательная колонка с URL
  • Необязательные колонки: ID, Model, Barcode
  • Полные URL (http:// или https://)
  • Могут содержать параметры запроса
  • Невалидные URL будут пропущены

Особенности

  • Сохранение параметра sku для Yandex Market
  • Поддержка дополнительных колонок (ID, Model, Barcode)
  • Экспорт в CSV и Excel

Пример обработки

Исходная URL:
https://example.com/product?utm_source=google&utm_medium=cpc&ref=123
Очищенная URL:
https://example.com/product

Как это работает

  1. Загрузите файл с URL
  2. Система проанализирует структуру
  3. Выберите колонки для обработки
  4. Запустите очистку URL
  5. Скачайте обработанный файл
