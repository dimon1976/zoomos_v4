Особенности
  • Сохранение параметра sku для Yandex Market
  • Поддержка дополнительных колонок (ID, Model, Barcode)
  • Экспорт в CSV и Excel
Шаг 1: Загрузка файла
Перетащите файл сюда или нажмите для выбора

Поддерживаемые форматы: Excel (.xlsx, .xls), CSV (.csv), TXT (.txt)

Требования к файлу
Структура файла
  • Первая строка - заголовки колонок
  • Обязательная колонка с URL
  • Необязательные колонки: ID, Model, Barcode
Формат URL
  • Полные URL (http:// или https://)
  • Могут содержать параметры запроса
  • Невалидные URL будут пропущены
Результат
  • CSV или Excel файл
  • Исходная URL | Очищенная URL
  • Дополнительные колонки при наличии
