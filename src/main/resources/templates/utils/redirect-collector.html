Сбор финальных URL после редиректов

Утилита проходит по каждой ссылке из файла, следует всем редиректам и возвращает финальный URL. Полезно для получения реальных адресов сокращённых ссылок.

Внимание: Обработка может занять значительное время, так как выполняются HTTP запросы к каждой ссылке. Рекомендуется использовать небольшие файлы (до 100 ссылок).
