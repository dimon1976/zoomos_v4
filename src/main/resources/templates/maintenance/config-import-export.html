Скачайте текущую конфигурацию в формате JSON для переноса на другой сервер.

Файл: zoomos-config-YYYY-MM-DD.json

Импорт конфигурации

Загрузите JSON-файл конфигурации. Существующие записи будут обновлены (merge), новые — добавлены. Расписания при импорте всегда отключены.

Шаг 1 из 3 — выберите файл

Что импортировать:

Шаг 2 из 3 — просмотр изменений
Секция Новых Обновить
Шаг 3 из 3 — результат
Обработка...
Как работает импорт: Существующие записи обновляются по бизнес-ключу (имя клиента, имя шаблона, имя магазина, название сайта). Новые записи создаются. Поля расписаний при импорте устанавливаются в отключено — включите их вручную после проверки. Добавление новых полей в сущности не ломает совместимость JSON-файлов.
