Анализ статистики позволяет отслеживать изменения в данных экспорта и выявлять аномалии.

Анализ статистики экспорта
Возможности анализа:
  • Сравнение данных экспорта между операциями
  • Выявление критических изменений в статистике
  • Анализ отклонений по настраиваемым порогам
  • Детальные метрики по полям и записям
  • Экспорт результатов анализа в Excel
Начать анализ
Как работает анализ:
  1. Выберите шаблон экспорта с включенной статистикой
  2. Укажите операции для сравнения
  3. Настройте пороги предупреждений
  4. Получите детальный отчет об изменениях
Внимание: Для анализа статистики необходимо создать шаблоны экспорта с включенной опцией сбора статистики.
